Download the ZIP from GitHub Releases and unpack it anywhere.

2.

Run PC_Workman_HCK.exe — a diagnostic console appears for a moment and hides itself once the UI is up.

Security & trust

Every release is built from the public source and signed with Sigstore. Windows SmartScreen may still warn on first run — that's the reality of indie software without a paid EV certificate. Verification steps and the full audit live in the Security Report; if in doubt, read the source code yourself.
