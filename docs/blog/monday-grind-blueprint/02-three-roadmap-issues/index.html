Three Roadmap Issues, Week One at a New Job

Context beats raw data. "epicgames.exe taking 40% CPU" looks like a virus until you know it's the launcher.

By Marcin Firmuga·2026-04-06·4 min read·Monday Grind Blueprint #2

Three issues from the roadmap to v1.7.8 this week - and the honest backdrop of a brand-new retail job eating most of my energy.

The three

  • Tests, discussions, packaging - a community-raised GitHub issue. Real coverage, not decoration.
  • Dashboard button redesign - they worked but still looked like 2015.
  • hck_GPT cleanup - performance pass and possible file consolidation.

Why context is the whole project

Last weekend's process library already changed how the app feels. Before: a user sees epicgames.exe - 40% CPU and thinks "virus, kill it?" After: hover, see it's the Epic Games Launcher, safe, typical. Context instead of raw data - the thesis the whole app is built on.
